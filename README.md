
# 🚀 J-log: 极简 GitHub Gist 个人博客系统

J-log 是一个基于 GitHub Gist API 开发的轻量级、响应式个人博客系统。无需数据库，无需服务器，只要有一个 GitHub Token，即可拥有一个功能完备、支持 Markdown 渲染和在线编辑的技术博客。

![](https://cdn.jsdelivr.net/gh/baidu8/J-log@main/Snipaste_2026-02-05_23-32-41.png)

## ✨ 核心特性
* 📦 Gist 驱动：全站内容存储在 GitHub Gist，安全、稳定且天然支持版本管理。

* ✍️ 沉浸式编辑器：

* 快捷工具栏：支持 H1-H3、粗体、引用、视频嵌入等一键插入。

* 📸 摄影集模式：内置 HTML 栅格布局，轻松实现双栏瀑布流图片展示。

* 实时预览：编辑时同步渲染 Markdown 效果，所见即所得。

## 🛡️ 内容保险：

* 防丢失提醒：刷新或关闭页面时自动弹出保存提醒，保护创作心血。

* 📱 深度适配：完美适配移动端，弹窗、菜单、工具栏均针对手机触控优化。

* 🔍 全量搜索：支持对所有 Gist 标题和文件名的即时模糊搜索。

* 📊 活跃度统计：内置基于 ECharts 的发布活跃度热力图。

* 💬 评论系统：集成 Giscus，利用 GitHub Discussions 托管评论。

## 🛠️ 技术栈
* Frontend: Vanilla JS (原生 JavaScript)

* Styling: CSS3 (Flexbox & Grid)

* Markdown: Marked.js

* Highlight: Prism.js

* Charts: ECharts
